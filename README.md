# Number_Guessing_CLI
A simple CLI-based Number Guessing Game built using Java.
The program generates a random number between 1 and 100, and the player must guess it within a limited number of attempts based on the selected difficulty level.

This project demonstrates Java fundamentals, including loops, conditionals, random number generation, user input handling, and basic game logic.
🎲 Random number generation between 1–100

🎚 Difficulty levels

Easy → 10 chances

Medium → 5 chances

Hard → 3 chances
